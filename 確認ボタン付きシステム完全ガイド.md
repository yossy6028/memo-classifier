## 🚀 使用方法

### 1️⃣ **ConfirmMemo.app**（推奨）

```
🖱️ アプリをクリック
📝 「タイトル | 内容」入力
👀 プレビュー表示
🎯 確認ボタンで選択：
   • Obsidianに送信
   • 編集
   • キャンセル
✅ 送信完了
```

**実例**：
```
入力: ObsidianとCursor連携 | 確認ボタン付きアプリを作成しました

プレビュー表示:
📋 タイトル: ObsidianとCursor連携
📂 カテゴリ: tech
🏷️ タグ: #Tech, #技術
📄 内容: 確認ボタン付きアプリを作成しました
💾 保存先: tech/ObsidianとCursor連携_0619.md

選択: [Obsidianに送信] [編集] [キャンセル]
```

### 2️⃣ **AdvancedMemo.app**（全機能版）

```
🖱️ アプリをクリック
📋 メニュー選択：
   • 確認付きメモ
   • 通常メモ（即座送信）
   • 一時保存管理
🎯 選択した機能を実行
```

### 3️⃣ **ターミナル版**

```bash
# 確認付きメモ作成
./quick_memo_with_confirm.py "タイトル" "内容"

# 対話モード
./quick_memo_with_confirm.py -i

# 一時保存ファイル一覧
./quick_memo_with_confirm.py --temp-list

# 一時保存ファイル送信
./quick_memo_with_confirm.py --send-temp
```

## 🎯 確認システムの流れ

### **Step 1: メモ入力**
```
📝 タイトル | 内容
例: Python学習 | デコレーターの使い方を理解した
```

### **Step 2: プレビュー表示**
```
============================================================
📝 メモプレビュー
============================================================
📋 タイトル: Python学習
📂 カテゴリ: tech
🏷️ タグ: #Tech, #技術
📄 内容:
----------------------------------------
デコレーターの使い方を理解した
----------------------------------------
💾 保存先: tech/Python学習_0619.md
============================================================
```

### **Step 3: 確認選択**
```
🎯 次の操作を選択してください:
  [s] Obsidianに送信
  [e] 編集
  [t] 一時保存（後で送信）
  [c] キャンセル

選択 (s/e/t/c): 
```

### **Step 4: アクション実行**

#### 🚀 **[s] Obsidianに送信**
```
✅ Obsidianに保存完了: tech/Python学習_0619.md
```

#### ✏️ **[e] 編集**
```
✏️ メモ編集モード
📋 タイトル [Python学習]: Python基礎学習
📄 内容 (現在: デコレーターの使い方を理解した...)
新しい内容を入力 (空行でEnter):
デコレーターとプロパティの使い方を理解した

再度プレビュー表示 → 確認選択
```

#### 📂 **[t] 一時保存**
```
📝 一時保存: /tmp/quick_memo/Python学習_0619_temp.md
```

#### ❌ **[c] キャンセル**
```
❌ メモ作成をキャンセルしました
```

## 📊 使い分けガイド

### 🎯 **ConfirmMemo.app** - 普段使い
- **用途**: 日常的なメモ作成
- **特徴**: 確認してから送信
- **シーン**: 学習記録、アイデア、会議メモ

### ⚡ **SimpleMemo.app** - 緊急時
- **用途**: 即座にメモを取りたい
- **特徴**: 確認なしで自動送信
- **シーン**: 電話メモ、思いつき、リマインダー

### 🔧 **AdvancedMemo.app** - 多機能
- **用途**: 状況に応じて機能選択
- **特徴**: メニューから機能選択
- **シーン**: 一時保存管理、機能試用

### 💻 **QuickTerminal.app** - 開発者
- **用途**: 複数メモの連続作成
- **特徴**: ターミナル対話モード
- **シーン**: プログラミング中、学習中
